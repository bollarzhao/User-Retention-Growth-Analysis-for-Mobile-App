# Mobile User Behavior Analysis & Prediction 移动设备用户行为分析与预测

随着智能手机的普及，理解用户的设备使用行为对于优化应用体验和产品设计具有重要意义。本项目通过分析移动设备用户行为数据，探索不同用户群体的使用模式，并构建机器学习模型预测用户行为类别。

数据集包含 700 个用户样本，包括应用使用时间、屏幕使用时间、电池消耗、数据流量等关键指标。

# Project Aims
This project analyze user behavior data, aims in answering following problems: 
  - 1️⃣ 不同用户群体的使用模式有什么区别
  - 2️⃣ 哪些因素最影响用户使用强度
  - 3️⃣ 能否通过机器学习预测用户行为类型

# Data description:
 The raw dataset is free open-access dataset from kaggle: https://www.kaggle.com/datasets/valakhorasani/mobile-device-usage-and-user-behavior-dataset/data
 
 You can also find the raw dataset under data branches named :"user_behavior_dataset.csv"

 Variables included: User_ID, Device_Model, Operating_System, App_Usage_Time, Screen_On_Time, Battery_Drain, Number_of_Apps_Installed, Data_Usage, Age, Gender, User_Behavior_Class
