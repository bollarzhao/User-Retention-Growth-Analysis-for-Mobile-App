# Mobile User Behavior Analysis & Prediction 移动设备用户行为分析与预测

## Project overview
This project analyzes smartphone usage patterns and predicts user behavior intensity using machine learning techniques. 

Using a dataset of 700 mobile users, the project explores how different usage metrics such as app usage time, screen-on time, battery consumption, and data usage relate to overall smartphone behavior patterns.

The analysis combinedexploratory data analysis, machine learning models, explainable AI techniques, clustering methods, and dimensionality reduction to better understand user behavior.

### Smartphone usage behavior varies significantly across users. Understanding these patterns can help:
 - improve mobile app design
 - optimize device performance
 - better understand digital habits

This project aims to:
  - explore smart phone usage data
  - predict user hebavior intensity
  - identify key drivers of smartphone engagement
  - segment users based on usage patterns

## Data description:
The dataset contains 700 user records with information about smartphone usage patterns. 
* Data source:
      The raw dataset is free open-access dataset from kaggle: https://www.kaggle.com/datasets/valakhorasani/mobile-device-usage-and-user-behavior-dataset/data
 
 You can also find the raw dataset under data branches named :"user_behavior_dataset.csv"

### Key Variables

| Variable | Description |
|---------|-------------|
| User ID | Unique identifier for each user |
| Device Model | Model of the smartphone |
| Operating System | Android or iOS |
| App Usage Time (min/day) | Daily time spent using apps |
| Screen On Time (hours/day) | Average screen active time per day |
| Battery Drain (mAh/day) | Daily battery consumption |
| Number of Apps Installed | Total number of apps installed |
| Data Usage (MB/day) | Daily mobile data consumption |
| Age | Age of the user |
| Gender | Gender of the user |
| User Behavior Class | Behavioral classification (1–5) representing increasing usage intensity |
